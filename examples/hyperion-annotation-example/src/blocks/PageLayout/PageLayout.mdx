# Page Layout
This is the main page layout, this is where most changes will be made in the 
future to make a responsive version of this site. It's important that tablets
and phones are able to view and hopefully contribute to these IIIF resources.

ordering of the elements can be driven by accessibility to allow a more logical
keyboard navigation experience.
